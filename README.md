# usage

```
git clone v1.2.0 https://github.com/stackr23/react23 --depth 1
cd react23
npm install
npm start 
```

## __npm scripts:__  
__start__ - starts webpack devServer
__build__ - builds webpack (production)
__lint__ - runs eslint
__test__ - runs gulp 'test'-task
